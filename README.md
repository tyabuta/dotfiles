
## Quick start

1. Setup dotfiles:

    ```
    $ git clone https://github.com/tyabuta/dotfiles.git
    $ cd dotfiles
    $ ./setup.sh
    ```
2. Install Vim Plugins

    ```
    $ git submodule init
    $ git submodule update
    ```
    vimエディターで、:NeoBundleInstall!


3. Bash Reboot
    
    .profileを有効にする為、ログアウトしbashを起動し直してください。
