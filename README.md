
## Quick start

1. Setup dotfiles:

    ```
    $ git clone --recursive https://github.com/tyabuta/dotfiles.git
    $ cd dotfiles
    $ ./install.sh
    ```

2. Bash reboot or reload
    
    .profileを有効にする為、ログアウトしbashを起動し直すか、
    .profileをsource(.)コマンドで読み直してください。

    ```
    $ . ~/.profile
    ```
